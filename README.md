# DTSW

DTSW 全称 `Does Trojan Still Work?`，是一个基于 Xray 的 Trojan 代理服务管理工具，灵感来源于 `easytrojan`，但只支持用户自有域名，不依赖免费共享域名。

当前实现使用：

- Go 控制平面（`dtsw`）
- Xray 作为 Trojan 运行时
- `acme.sh` 作为 ACME 客户端，支持 `Let's Encrypt` 和 `ZeroSSL`
- Caddy 静态网站作为回落服务
- `systemd` 单元管理运行时、回落服务和自动续签

## 一键安装

```bash
curl -fsSL https://github.com/zhaodengfeng/DTSW/releases/latest/download/install.sh | bash
```

## 交互界面

安装完成后直接进入管理面板，之后执行 `dtsw` 即可重新打开。

**启动菜单**

```
╔══════════════════════════════════════╗
║              DTSW 交互菜单               ║
╚══════════════════════════════════════╝

  配置文件:           /etc/dtsw/config.json
  域名:              trojan.example.com
  用户数:            1

  1) 打开管理面板
  2) 使用已保存配置安装或修复
  3) 查看客户端配置
  4) 重新运行引导安装
  0) 退出
```

**管理面板**

```
╔══════════════════════════════════════╗
║              DTSW 管理面板               ║
╚══════════════════════════════════════╝

  域名:              trojan.example.com
  配置 Xray 版本:    v26.1.13
  已安装 Xray 版本:  v26.1.13
  最新 Xray 版本:    v26.2.0
  用户数:            2

  1) 查看客户端配置
  2) 查看运行状态

  4) 升级 Xray 到最新版
  5) 恢复 Xray 到配置版本
  6) 续签 TLS 证书
  7) 用户与流量管理
  8) 卸载 DTSW
  0) 退出
```
